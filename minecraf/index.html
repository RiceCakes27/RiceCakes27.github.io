To join the server and become the dragon you were always meant to be, simply download this CurseForge mod pack:

Click image

Once you have downloaded the mod pack, follow these steps to install it:

  1. Download CurseForge and go through the installation, if you don't already have it.
  2. Click "Create Custom Profile" and select "import a previously created profile".
  3. Select the DragonLand ZIP file you downloaded and click "Open".
  4. Now just click play! And join the server in multiplayer.
